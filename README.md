# TinySprites Pages

This directory contains the web pages for the TinySprites project, designed to work with GitHub Pages.

## Pages

### `index.html` - Sprite Editor

The main editor page where users can:

- Create custom sprites with a visual grid editor
- Set dimensions and color palettes
- Paint sprites pixel by pixel
- Generate packed codes for sharing
- Test sprites in the demo page

### `demo.html` - Sprite Demo

The demo page where users can:

- Input packed codes to see sprites rendered
- Browse built-in example sprites
- Click examples to load them into the decoder
- Learn about the packed code format

## Features

- **Visual Sprite Editor**: Click-to-paint grid interface
- **Real-time Preview**: See sprites as you create them
- **Packed Code Generation**: Get shareable sprite codes instantly
- **Example Library**: Built-in sprites to learn from
- **GitHub Pages Ready**: Works perfectly when deployed

## Navigation

Both pages include navigation between the Editor and Demo, making it easy to switch between creating and viewing sprites.

## Usage

1. **Create Sprites**: Use the Editor page to design custom sprites
2. **Share Codes**: Copy the generated packed codes
3. **View Sprites**: Use the Demo page to decode and display any packed code
4. **Learn**: Browse examples to understand the format

## GitHub Pages Deployment

These pages are automatically deployed to GitHub Pages when pushed to the main branch. The workflow deploys from this `pages/` directory.
